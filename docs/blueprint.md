# **App Name**: KisanAI: Empowering Farmers with AI

## Core Features:

- Dashboard with Weather & Quick Links: Displays a greeting, weather summary fetched via OpenWeather API, and quick links to essential features like Crop Advisory, Farm Planner, Soil Health, Satellite View, Market Prices, AI Assistant, Voice Assistant, Disease Detector, and Community. Icons with hover effects enhance the user experience.
- AI Disease Detector: Allows farmers to upload plant images, analyze symptoms, identify diseases using AI, suggest solutions, recommend pesticides, and provide preventive measures using Gemini API.
- AI Chatbot: Provides an AI Chatbot powered by Gemini API to answer any farming-related question in any language via text and image input, displayed in a green-themed chat UI.
- Voice Assistant: Enables farmers to speak their questions, which are converted to text and sent to the AI assistant (Gemini API), with the answer read out loud using text-to-speech.
- Farm Planner: Generates sowing, irrigation, fertilizer, and pest risk timelines based on land size, crop name, and sowing date inputs, displayed in a timeline + card format.
- Soil Health Analyzer: Analyzes soil health by either image upload or manual data input to show key parameters (N, P, K, pH, EC, Organic Carbon) with color-coded indicators and fertilizer recommendations.
- Satellite View: Utilizes NASA GIBS NDVI tiles to provide satellite imagery of farm locations, including historical NDVI, color legends, and farm boundary overlays.

## Style Guidelines:

- Primary color: Standard Green (#008000) to reflect the eco-friendly theme.
- Background color: Soft grey (#F0F0F0) provides a neutral, clean backdrop that enhances the green elements without overshadowing them.
- Accent color: Lime (#70E000) to highlight interactive elements and CTAs.
- Body and headline font: 'PT Sans', a humanist sans-serif, providing both modern look and a little warmth or personality; well-suited for headlines or body text.
- Farmer-friendly icons with a consistent, simple style to represent various tools and information.
- Modern, bright, eco-friendly design with smooth transitions, hover scale animations (1.03), rounded cards, soft shadows, glassmorphism for tool sections, sticky top header, and mobile-first responsiveness.
- Smooth transitions and hover scale animations (scale 1.03) to create an interactive and engaging user experience.
